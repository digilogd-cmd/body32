# BODY32

> Decode Your Body. Balance Your Life.

BODY32는 동양 철학에서 영감을 받은 글로벌 웰니스 성향 플랫폼입니다. 짧은 질문을 통해 32개 Guardian 중 하나를 만나고, 개인화된 BODY32 Passport를 만들 수 있는 제품을 개발합니다.

BODY32는 의료 진단이나 치료 서비스가 아닙니다.

## 현재 단계

제품·브랜드·디자인·캐릭터·알고리즘·기술 아키텍처 명세를 완료하고 웹 애플리케이션 기반을 구축하고 있습니다.

## 로컬 실행

```bash
npm install
npm run dev
```

품질 검사:

```bash
npm run lint
npm run typecheck
npm test
npm run build
```
