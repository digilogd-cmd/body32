# BODY32 MVP Release Checklist

Status: local pre-release baseline. Public deployment is intentionally not authorized yet.

## Product

- [x] Korean and English landing, quiz, result, Passport, and sharing flows
- [x] 20 versioned questions and deterministic 32-type calculation
- [x] 32 Guardian portraits and bilingual archetype guidance
- [x] Five-element Balance with plain-language explanation
- [x] Restart and home recovery paths

## Trust and privacy

- [x] No account, email, or profile details required
- [x] No raw answers or internal scores in sharing payloads
- [x] No server, localStorage, or sessionStorage persistence
- [x] Wellness—not medical diagnosis—boundary appears in core surfaces and Passport
- [x] No analytics or third-party tracking in the local MVP

## Accessibility and localization

- [x] Keyboard-operable quiz controls and deterministic focus movement
- [x] Skip navigation, live status messages, semantic progress, and reduced-motion support
- [x] Korean and English schemas are identical and automatically tested
- [x] Guardian meaning is available as text and never communicated by color alone
- [ ] Native editorial review before a public marketing launch

## Engineering

- [x] Lint, strict typecheck, automated tests, and production build
- [x] All 32 stable Guardian IDs resolve to content and portrait art
- [x] Runtime portrait sheets optimized to WebP
- [x] Local preview remains the approved review environment

## Required before public deployment

- [ ] Owner approval of final Korean and English copy
- [ ] Legal/privacy review appropriate to launch markets
- [ ] Final domain, hosting, and canonical URL decision
- [ ] Public share-link and social-preview configuration
- [ ] Production error monitoring and privacy-preserving analytics decision
- [ ] Explicit deployment authorization
