## Objective

Define a stable, explainable 32-type taxonomy and a unified Character Bible that can support result logic, localization, illustration production, Passport design, social assets, and future app experiences.

## Context

The approved concept requires 32 premium Guardian characters. Earlier planning proposed four constitution groups multiplied by eight energy groups, while also presenting five-element balance. Treating all of these as the same axis would create inconsistent or hardcoded mappings. This task separates the stable 32-type identity from the five-element balance visualization.

## Scope

- Define four non-medical Energy Rhythms.
- Define eight Guardian Archetypes.
- Define the 32 stable type IDs and working public names.
- Define how type identity relates to five-element balance.
- Define shared character art direction, anatomy, assets, production process, and QA.
- Define localization, accessibility, cultural safety, and versioning rules.

## Out of scope

- Final illustration generation
- Final scoring weights or quiz questions
- Clinical or traditional medicine equivalence
- Production JSON/TypeScript data
- Final type descriptions and recommendations
- 3D modeling, animation rigs, voice, merchandise, or NFT work

## Acceptance criteria

- Exactly 32 stable types are generated by a clear `4 × 8` model.
- Type identity does not claim medical diagnosis or map directly to a protected clinical label.
- Five-element balance remains a separate, explainable score vector.
- Every type has a stable machine ID, rhythm, archetype, and working public name.
- Shared visual rules make the 32 characters recognizable as one universe.
- Required asset variants and naming conventions are defined.
- Localization, accessibility, cultural review, and visual consistency checks are explicit.
- Working names can change without breaking stable IDs or stored results.

## Verification

- Confirm four unique rhythms and eight unique archetypes.
- Confirm the type registry contains 32 unique IDs and names.
- Check every rhythm/archetype pair appears exactly once.
- Run `git diff --check`.
- Review the diff for unrelated changes.

## Risks and decisions

- Public names are working names until linguistic and trademark review.
- Animal symbolism varies by culture; launch copy and imagery require cultural review.
- Five-element defaults must not predetermine a user's chart; TASK-003 will compute the score vector from answers.
- Illustration consistency cannot be guaranteed by prompts alone and requires visual QA.
