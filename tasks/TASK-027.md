# TASK-027 — Product Vision direction reset

## Objective

Restore BODY32's original platform ambition while correcting the current overemphasis on Guardian matching.

## Deliverables

- A standalone Product Vision source document.
- A useful Body Profile positioned ahead of Guardian identity.
- Defined roles for Eastern-wellness inspiration, MBTI context, personal data, deterministic scoring, and AI explanation.
- Clear non-diagnostic safety boundaries and measurable utility goals.

## Acceptance criteria

- [x] The core outcome is more useful than an animal match.
- [x] Guardian and Passport remain important IP layers.
- [x] Personal inputs have explicit purposes and limits.
- [x] AI and deterministic algorithm responsibilities are separated.
- [x] The next nine document revisions are ordered.

## Decision

The existing MVP is reclassified as a visual and technical prototype. Feature development pauses behind document realignment, beginning with the PRD.
